# E2E Test Commands Quick Reference

## Run Tests

```bash
# All tests
npm run test:e2e

# UI mode (interactive)
npm run test:e2e:ui

# Headed mode (see browser)
npm run test:e2e:headed

# Specific file
npx playwright test e2e/wallet-disconnect.spec.ts

# Specific test
npx playwright test -g "Dashboard shows connect"

# All browsers
npx playwright test --project=chromium --project=firefox --project=webkit

# Single browser
npx playwright test --project=chromium

# Mobile
npx playwright test --project="Mobile Chrome"
```

## Debug

```bash
# Debug mode (step through)
npx playwright test --debug

# Debug specific test
npx playwright test -g "Account page" --debug

# Headed + slow motion
npx playwright test --headed --slow-mo=1000

# Trace viewer (after failure)
npx playwright show-trace trace.zip
```

## Report

```bash
# Show last report
npx playwright show-report

# Generate report
npx playwright test --reporter=html
```

## Update

```bash
# Update snapshots
npx playwright test --update-snapshots

# Install browsers
npx playwright install

# Install with dependencies
npx playwright install --with-deps
```

## Watch

```bash
# Watch mode
npx playwright test --watch
```

## CI Mode

```bash
# Run as CI would
CI=true npm run test:e2e
```

## Filter

```bash
# By file name
npx playwright test wallet-disconnect

# By test name
npx playwright test -g "disconnect"

# Inverse grep (exclude)
npx playwright test -g-v "skip-this"
```

## Options

```bash
# Max failures
npx playwright test --max-failures=3

# Retries
npx playwright test --retries=2

# Workers (parallel)
npx playwright test --workers=4

# Timeout
npx playwright test --timeout=60000

# Headed + debug logs
npx playwright test --headed --debug
```
